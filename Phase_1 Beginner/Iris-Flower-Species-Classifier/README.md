# Iris Flower Species Classifier

This project is a beginner-friendly machine learning classification project built on Kaggle. The main objective is to classify iris flowers into three species — **Setosa**, **Versicolor**, and **Virginica** — using flower measurements such as sepal length, sepal width, petal length, and petal width.

Kaggle Notebook: [Iris Flower Species Classifier](https://www.kaggle.com/code/mnoumanrasheed/iris-flower-species-classifier)

---

## Project Overview

The Iris dataset is one of the most commonly used datasets for learning machine learning classification. In this project, the dataset is explored, cleaned, visualized, and used to train classification models. The workflow covers basic data analysis, feature understanding, model training, prediction, and performance evaluation.

This project is useful for understanding how machine learning models learn patterns from data and make predictions on unseen examples.

---

## Dataset

The project uses the famous **Iris Flower Dataset**, which contains measurements of iris flowers. Each flower belongs to one of three species.

### Features Used

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Variable

- Iris Species
  - Setosa
  - Versicolor
  - Virginica

## Project Workflow

1. Import required Python libraries
2. Load the Iris dataset
3. Explore the dataset using basic statistics
4. Check missing values and understand data structure
5. Visualize relationships between features
6. Prepare features and target variable
7. Split the dataset into training and testing sets
8. Train machine learning classification models
9. Make predictions on test data
10. Evaluate model performance using accuracy, confusion matrix, and classification report

---

## Machine Learning Models

The project may include classification models such as:

- K-Nearest Neighbors
- Decision Tree Classifier
- Logistic Regression
- Support Vector Machine

The purpose is to compare model performance and understand which model gives better results on the Iris dataset.
